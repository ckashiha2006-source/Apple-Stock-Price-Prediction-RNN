# Apple Stock Price Prediction using Recurrent Neural Network (RNN)

## 📌 Overview

This project demonstrates how a **Recurrent Neural Network (RNN)** can be used to predict Apple Inc. (AAPL) stock prices using historical stock market data.

The project follows a complete Deep Learning workflow, from data preprocessing to model training, prediction, evaluation, and visualization.

---

## 🎯 Objective

The goal of this project is to build a Deep Learning model capable of learning patterns from historical Apple stock prices and predicting future stock values.

---

## 📂 Dataset

**Dataset:** AAPL.csv

The dataset contains historical stock market information, including:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close Price
- Volume

1. Import Dataset
2. Data Cleaning
3. Feature Selection
4. Data Normalization
5. Create Time-Series Sequences
6. Build the RNN Model
7. Train the Model
8. Predict Stock Prices
9. Evaluate Model Performance
10. Visualize Actual vs Predicted Prices

---

## 🧠 Model Architecture

- Simple Recurrent Neural Network (RNN)
- Dense Output Layer
- Mean Squared Error (MSE) Loss
- Adam Optimizer

---

## 📊 Results

The trained RNN model learns temporal patterns from historical Apple stock prices and predicts future values. Performance is evaluated by comparing predicted prices with actual stock prices using graphical visualizations and regression metrics.
